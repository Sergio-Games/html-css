Trabalhando com CSS Externo

O que é?

O css inline consiste em utilizar as propiedades de css dentro de cada tag individualmente

assim eu posso mudar as propiedades dos meus conteudos do site, como por exemplo a cor deste parágrafo

Desvantagens

  • desorganização: Utilizando o css inline o seu código fica extremamente poluido e difícil de identificar o que é css e html no seu código
  • não praticidade: O principal malefício da utilização do css inline é a dificuldade de modificar propiedades de tags em geral, pois você teria que viajar de uma a uma para realizar tal ato, isso em códigos mais complexos iria levar muito tempo.
